NgRx Async Effects

This page demonstrates how NgRx handles async API calls with actions, effects, reducers, and selectors.

Fake Login Flow

@if (loading$ | async) {

Loading...

} @if (error$ | async; as error) {

{{ error }}

} @if (user$ | async; as user) {

User: {{ user.name }}

Email: {{ user.email }}

Role: {{ user.role }}

} @if (token$ | async; as token) {

JWT:

{{ token }}
}

Interview Notes

  • login action represents the start of the API request.
  • AuthEffects calls POST /api/login.
  • loginSuccess stores the token and user.
  • loginFailure stores the error message.
  • Selectors expose loading, user, token, and error state to the component.

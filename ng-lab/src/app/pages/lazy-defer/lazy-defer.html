Lazy Loading + @defer

This page compares route-level lazy loading with template-level deferred loading.

Route Lazy Loading

This page is lazy-loaded by the router using loadComponent in app.routes.ts. 

loadComponent: () => import('./pages/lazy-defer/lazy-defer')

Template Defer

The chart below is not rendered until the button is clicked.

@defer (when shouldShowChart()) { } @placeholder {

Chart has not been loaded yet.

} @loading {

Loading chart...

}

Notes

  • loadComponent lazy-loads an entire route component.
  • @defer lazy-loads part of a template.
  • @placeholder shows before the deferred block starts loading.
  • @loading shows while the deferred block is loading.
